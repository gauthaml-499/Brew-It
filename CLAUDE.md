# Brew-it

## Skill priorities

For any UI/frontend design, styling, or review work in this project, prioritize these skills (in this order) over general-purpose approaches:

1. **frontend-design** — for aesthetic direction, typography, and layout choices when building or reshaping UI.
2. **ui-ux-pro-max** — for concrete implementation: component styles, palettes, font pairings, and stack-specific patterns.
3. **impeccable** — for reviewing, critiquing, and polishing UI (hierarchy, accessibility, spacing, responsiveness, states).

Invoke the relevant skill via the Skill tool before starting frontend/UI tasks rather than defaulting to ad hoc styling decisions.
